To Install:
	Simply put the folder of what you want from the "DISABLED" folder into the "override" folder.
	To remove it, just put the folder back into "DISABLED".
	
	
	Example:
	
	overrides
        └── DISABLED
		├── Transparent-Viewmodels
		└── Transparent-3D-Class
			
					|
					|
					|
					|
					|
					V
					
	overrides
        └── DISABLED
		└── Transparent-3D-Class
	└──Transparent-Viewmodels
